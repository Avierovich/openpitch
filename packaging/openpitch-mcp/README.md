# OpenPitch MCP server

Launcher package for the [OpenPitch](https://github.com/Avierovich/openpitch) MCP server —
open, real-time, fully-sourced intelligence on AI startups. Run it with:

```bash
uvx openpitch-mcp
```

All functionality lives in the [`openpitch`](https://pypi.org/project/openpitch/) package;
this wrapper just makes the zero-install command work.
